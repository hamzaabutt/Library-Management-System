# Library Management System

This project is a console-based Library Management System developed in C++.
It allows librarians or administrators to manage books, users, and transactions through a simple text-based interface.

# Features

- Add, view, and manage books
- Add and manage library users
- Borrowing and returning of books
- Transaction history with timestamps
- Search books by Title / Author
- Search users by Name
- Sort books by Title or Author
- Save and load data using text files

# Technologies Used

- C++ (Procedural Programming)
- File Handling
- Arrays and Structures
- Bubble Sort
- String Processing

# Files Included

main.cpp — Complete source code
books.txt — Stored book records
users.txt — Stored user records
transactions.txt — Stored transaction history
